# testing-assets-for-TSF

A small collection of Ardiuno scripts I made to measure variables for a Toronto Science Fair Project (2026)
These variables include:
- CH4 ppm
- RPM of a turbine
- Pressure, temperature, and humidity inside a sealed container

Our project generates electricity from organic matter using anaerobic bacteria digestion. Methanogens, a type of bacteria, break down waste and produce methane biogas. The methane is filtered and oxidized, then burned to heat a flash boiler. This produces steam, which spins a turbine connected to a brushless motor, generating electricity. Our system demonstrates how a biological process can accomplish two goals: reducing waste while generating renewable energy.
